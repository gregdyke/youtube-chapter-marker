Chapter Marker Player Example

You write: 

<iframe width="640" height="390"
src="//www.youtube.com/embed/M7lc1UVf-VE?enablejsapi=1"
frameborder="0"></iframe>
<div class="chapters">
0:00 start<br>
0m50 next<br>
@1:35 another<br>
10:40s yet another chapter<br>
15mn30 last<br>
1:05:10 this one doesn't exist but shows how to use hours
</div>

It produces:
0:00 start
0m50 next
@1:35 another
10:40s yet another chapter
15mn30 last
1:05:10 this one doesn't exist but shows how to use hours

Usage notes:

  • Include stylesheet: <link rel="stylesheet" href="chapter_marker_creator.css"> chapter_marker_creator.css
  • Include javascript: <script type="text/javascript" src="chapter_marker_creator.js"></script> chapter_marker_creator.js
  • IFrame youtube urls should include ?enablejsapi=1 in the url (see above)
  • Each IFrame with chapter indications should be immediately followed by <div class="chapters"></div>
  • Each chapter description should be separated from the next by <br>
  • Next plan is to make the entry of the chapters more flexible, if necessary (e.g. in paragraphs, or whatever wordpress autogenerates by default).
